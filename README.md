# java-cli-gradle-findbugs-jbehave-testng-hello-world

## Description
A POC app using testng,
findbugs, jbehave framework.

## Tech stack
- java
- gradle
  - testng
  - findbugs
  - jbehave

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
